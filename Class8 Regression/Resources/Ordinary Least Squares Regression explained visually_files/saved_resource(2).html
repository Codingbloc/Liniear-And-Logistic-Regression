            Rational E.T. Ken Kahn a year ago

            In the last example, each point can move independently along the x-axis, affecting either x1 on the first panel or x2 in the second one. When moving a point vertically, then you're affecting variable y, which has to be the same on both panel. So only when moving vertically do you affect both panels at once. If you move horizontally, you only affect the variable corresponding to that specific panel. x1 and x2 are indeed independent.

                                Tom Smith Summer Cutrer Tierno a year ago

                                They are not the same. The error (or disturbance) of an observed value is the deviation of the observed value from the (unobservable) true value of a quantity of interest (for example, a population mean), and the residual of an observed value is the difference between the observed value and the estimated value of the quantity of interest (for example, a sample mean).
